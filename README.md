# NNS Bass Trainer

A web-based drill app for a 5-string electric bass player learning to internalize the Nashville Number System across all 12 keys.

## What it does

Flashcard-style practice: you're shown a Nashville number (1–7) and a key, and you identify the correct note name from four multiple choice options. Wrong answers are re-queued and shown again later in the session. Each session is 10 questions, ending with a score and per-number breakdown.

## Stack

- React + TypeScript
- Vite
- Tailwind CSS v4

## Running locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).
